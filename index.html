História dos computadores

Uma das maiores invenções que podemos ver até nos dias de hoje é o computador, criado para ser um meio de comunicação militar, contudo, sua necessidade se mostrou muito mais ampla do que somente a guerra, a criação dos pontos de redes unicas e a criação de redes compartilhadas como a internet, nós possibilitou uma comunicação muito mais efetiva e muito mais promissora tanto tecnologicamente quanto a comunicação global, avançando para uma sociedade muito mais tecnologia e eficiente. As primeiras maquinas desenvolvidas não dependiam de energia para funcionar e só mente a lógica matematica, como maquinas que conseguiam ler pontos pontilhados e realizar calculos ou tocar musica em um piano,

O que são Clients?

A definição de cliente é a maquina que solicita os dados, navega pela rede ou cria documentos, é a ponta final de um sistema que conversa com o servidor, ela realiza solicitações e aguarda uma resposta do servidor, como verificar arquivos, imagens ou acessar uma pagina WEB

O que são Servers?

Servidores são computadores que armazenam informações, como, dados de clientes, jogos, imagens e uma infinidade de informações, A função principal de um servidor é de receber solicitações e enviar as respostas ao cliente, de modo que ele pode lidar com diversidades de informações simultâneas como recebimento demasiado de requisiçoes e ter uma demanda de respostas alta, dependendo do tipo de servidor que é projetado tanto fisicamente quanto em sua estrutura de software.

Algumas informações sobre servidores

  • Um servidor é basicamente um computador dedicado ao armazenamento de dados.
  • Um servidor pode ser tanto local quanto remoto

Como é uma estrutura básica de um HTML?

Uma estrutura de básica de uma página consiste em uma sequencia de Tags tais como:

  1. html - É o inicio de uma estrutura básica de html, aonde é composto todo a marcação de códigos
  2. body- É aonde fica todo o "corpo" do seu código aonde será visivel ao usúario
  3. h1 á h6 - São as formas de titulos como você vê escritos nesse site como podemos mostrar dessa forma:

    • h1

    • h2

    • h3

    • h4

    • h5
  4. Footer- é aonde fica o fim da página

Entendendo a diferença entre listas ordenadas e não ordenadas

O que são as listas ordenadas
São listas que possuem numeração espefica, como podemos definir graus de prioridade, passo a passo como nesse exemplo:

  1. Acordar
  2. Tomar café
  3. Tomar banho

Esse tipo de lista

Listas não ordenadas
As listas não ordenadas são listas que não tem a necessidade de serem numeradas ou tem um grau especifico de prioridade, são apenas listas como no exemplo a seguir:
  • Carros antigos
  • Limpeza de quarto
  • Paisagens belas

Esse é um tipo de lista que não possui ordem, porém pode ter contexto dependendo do que se for colocar na lista.
